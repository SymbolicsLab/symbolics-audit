## Advances integrated into vault

### 1. Maximal resilience → phenomenology of necessity (NEW)
Modus ponens stabilizes under every conservative, distinction-preserving policy. This maximal resilience produces the felt compulsion of deductive inference. Where resilience is partial, inference feels like a choice.
- **Added to**: Vault 02_Operations, after policy-fixity gradient table

### 2. Dissolution of inference as category (SHARPENED)
"Inference" is not a natural kind but a region on the constraint-fixity spectrum. Articulability, not ontological privilege.
- **Added to**: Vault 02_Operations, extending existing gradient paragraph

### 3. Two demarcation criteria: inference vs. physical settling (NEW)
Endogenous revisability + commitment/realization gap. Both required. Ball-in-valley fails both. Places each case on the law/rule gradient without sharp threshold.
- **Added to**: Vault 02_Operations, new subsection "Inference vs. physical settling"

### 4. Failed inference vs. fabrication (NEW)
Misapplying a policy (failed inference, still inference) vs. generating without constraint (fabrication, not inference).
- **Added to**: Vault 02_Operations, within "Inference vs. physical settling" subsection

### 5. Coupled-system argument for AI (NEW)
LLM alone doesn't infer (no endogenous revisability). The coupled system (user + model + pipeline) does. Model = stabilization operator; ecology = revisability.
- **Added to**: Vault 02_Operations, within "Inference vs. physical settling" subsection
